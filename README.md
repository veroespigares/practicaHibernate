Descripción del CRUD resuelto con Hibernate:

Entidades:

- Consulta
- Paciente
- Síntoma



Con este CRUD se intenta controlar:

- Todos los datos de las consultas. (añadir,modificar,borrar)
- Relacionar los pacientes con una consulta, por lo tanto, cada consulta tendrá muchos pacientes
pero los pacientes solo tendrán una consulta.
- Gestionar la relación consulta-paciente.
- Todos los datos de los pacientes. (añadir,modificar,borrar)
- Todos los datos de los síntomas. (añadir,modificar,borrar)
- Relacionar los pacientes con los síntomas, por lo tanto, cada paciente tendrá muchos síntomas
y cada síntoma lo tendrán muchos pacientes.
- Gestionar la relación paciente-síntoma.
